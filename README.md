## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/taleshrocha/medium-like-app.git
   ```
2. Install npm packages
   ```sh
   npm install
   ```

## Usage

### Up and Running

Run the application in localhost with

```sh
npm run dev
```
